Stellar Insights - Astrology Landing Page Template Documentation

Thank you for choosing the Stellar Insights Astrology Landing Page Template! This documentation will guide you through the customization process, allowing you to tailor the template to match your brand and business needs.

Customization

1. Navbar Customization

To customize the navigation bar, follow these steps:

  • Open the HTML file containing the navbar section.
  • Locate the <title> tag and replace the text with your desired webpage title.
  • Replace the title icon with your own by placing the icon in the "images" folder and updating the href attribute of the <link> element.

2. Hero Section Customization

To customize the hero section, follow these steps:

  • Open the HTML file containing the hero section.
  • Modify the text content to align with your business message and offerings.
  • Change the background image by replacing the existing image in the "images" folder.

3. About Section Customization

To customize the about section, follow these steps:

  • Open the HTML file containing the about section.
  • Update the text content to describe your business and services.
  • Replace the business logo with your own by placing the logo image in the "images" folder.
  • Update the social media links to connect with your business profiles.

4. Services Section Customization

To customize the services section, follow these steps:

  • Open the HTML file containing the services section.
  • Modify the text content to showcase your unique astrology services.
  • Customize icons by replacing existing icons in the "images/icons" folder.

5. Featured Astrologers Customization

To customize the featured astrologers section, follow these steps:

  • Open the HTML file containing the featured astrologers section.
  • Update the text content and social media links for each featured astrologer.
  • Replace thumbnail images by placing new images in the "images/astrologers" folder.

6. Testimonials Customization

To customize the testimonials section, follow these steps:

  • Open the HTML file containing the testimonials section.
  • Modify the text content to showcase positive feedback from your clients.

7. Contact Form Customization

To customize the contact form, follow these steps:

  • Open the HTML file containing the contact form.
  • Change the action attribute of the <form> element to your form backend URL. You can use free services like Formspree for this purpose.
  • Update the phone number, email, and social media links to reflect your business information.
